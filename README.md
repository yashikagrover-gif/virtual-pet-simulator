# 🐾 Virtual Pet Simulator

A Python-based Virtual Pet Simulator developed as part of my internship assignment.

## Features

- Choose your pet (Dog, Cat, Rabbit, Bird)
- Feed your pet
- Play with your pet
- Give toys
- Give medicine
- Track happiness
- Track hunger
- Energy system
- XP & Level system
- Random events
- Save & Load game using JSON
- Pet status dashboard
- Final report

## Technologies Used

- Python
- Jupyter Notebook
- JSON
- Random Module

## How to Run

1. Open the notebook in Jupyter Notebook.
2. Run all cells.
3. Follow the on-screen menu.
